# driver backend
```
Node.js v18.19.0 + Express + Typescript
```

### For development
```
npm run start:dev
```

### For production
```
npm run start:prod
```

### Folder
asset: store driver files<br>
backend_dist: backend compiled code<br>
frontend_dist: frontend compiled code<br>
log: runtime log<br>
temp: store temp files when receive upload
